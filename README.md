# MaNNography

## Dataset Builder

Then, generate the dataset run:

```python
python main.py path/to/dataset
```


`path/to/dataset` should be the parent directory of the `classes`, for example:

    dataset/
    ├── normals/
    ├── benign_without_callbacks/
    ├── cancers/
    └── benigns/

Here, the path would end in `dataset/`.
